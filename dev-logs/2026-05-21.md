### 阶段 2 ✅：明信片纸质化重设计
- 明信片从暗色玻璃 → 暖色纸质纹理
- 正面：旧信纸 + 油画布风格
  - 底色：陈年羊皮纸 `#f3e5c8 → #ddc080`（暖黄旧调）
  - 褪色斑痕：多层径向渐变模拟岁月痕迹
  - 画布颗粒：`repeating-conic-gradient` 密纹肌理
  - 上传照片后，纹理叠加在照片上层（`::after`）
- 焦边揉皱纸升级（第三次迭代）：
  - SVG feTurbulence + feDiffuseLighting 生成真实褶皱凹凸光影
  - feDisplacementMap 让边缘不规则毛糙（±5px 位移）
  - 焦边晕染：多层 `radial-gradient` 中心米黄 → 外圈深褐（`#f7eddc → #6b4520 → #4a2a0c`）
  - 不规则圆角 `border-radius: 44% 40% 48% 42% / 43% 47% 40% 45%`
  - 哑光无光泽，接近真实揉皱牛皮纸
- 背面：编辑信纸风格
  - 可编辑字段：标题、日期、心得笔记
  - 虚线边框输入框 + 聚焦实线边框
  - 所有编辑实时保存到 localStorage
- 正面：文字可编辑

  - placeholder 显示默认中英文值，编辑后替换
