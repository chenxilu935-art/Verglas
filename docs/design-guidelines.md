# 设计规范

## 整体风格
- **暗色编辑志风 + 海洋漂流主题**
- 克制用色：深海蓝为底，暖白/纸色为内容载体
- 大留白、不对称排版、强节奏感标题

## 色彩与氛围
- 背景：真实海洋照片 + 暗角渐变（深海蓝 → 墨蓝 → 黑）
- 内容卡片：暖纸色 / 半透明玻璃（与海洋背景形成冷暖对比）
- 强调色：暖金 / 琥珀（点缀用，少量克制使用）
- 辅助色：淡灰、白色、低饱和玻璃白
- 质感：轻颗粒背景、纸纹理、柔和光晕

## 字体
- 标题：楷体（KaiTi, serif）— 大字号、分层排版
- 正文：Arial / sans-serif
- 兼容处理：如果系统无楷体，回退到 `serif`

## 布局
- 编辑志风格：大标题 + 宽留白 + 不对称错落
- 大屏为错落网格，小屏为单列堆叠
- 顶部导航 sticky + 吸附效果
- 内容区最大宽度限制，两侧留白

## 组件风格

### 明信片（Postcard）
- 纸质纹理：CSS `filter` / 噪点叠加，模拟真实纸张
- 暖白/米色底，微卷边阴影
- 缓缓漂浮动画：从远处漂来、不规则摇摆
- 翻转交互：hover 时 3D 翻转显示背面内容
- 每张明信片承载不同内容入口（介绍、故事、经历）

### 内容卡片
- 半透明玻璃拟态或暖纸色卡片
- 微颗粒纹理叠加
- 柔和阴影，边缘渐变

### 按钮
- 渐变边框、悬停亮度提升
- 大留白按钮（编辑志风格）

## 动效与交互

### 电影感滚动叙事
- 元素进入：staggered reveal（错落淡入 + 上移）
- Scroll-triggered：滚动到特定位置触发动画序列
- 每个区块有独立的"登场"节奏
- 过渡：`ease-out`、`duration-500`~`700`

### 漂流明信片
- 主动画：从画面外或远方缓缓漂入
- 浮动状态：不规则摇摆 + 微旋转 + 缓慢位移
- 每张明信片有不同的动画节奏和相位

### CSS 3D 效果
- 卡片翻转：`perspective` + `rotateY`（hover 触发）
- 景深感：不同 z-index 层次有不同的视差速度
- 视差层次：海洋背景层 / 明信片层 / 内容层，滚动时以不同速度移动

### 微交互
- 悬停：轻微放大、阴影变化、3D 倾斜
- 弹性：小范围 `scale` / `translateY` 动画

## 内容交互
- 首页同时支持中英文切换按钮
- 简历页提供独立语言切换开关
- 页面内显示编辑按钮，进入"编辑模式"后可直接修改文本
