# 设计规范

## 色彩与氛围
- 主色：蓝色与紫色渐变
- 辅助色：淡灰、白色、半透明玻璃白
- 质感：轻颗粒背景、柔和光晕、高级感渐变

## 字体
- 标题：楷体（优先使用系统楷体）
- 正文：Arial / sans-serif
- 兼容处理：如果系统无楷体，可使用类似 `KaiTi, serif`

## 布局
- 使用流体布局，保持卡片在不同屏幕间自然伸缩
- 大屏为错落网格，小屏为单列堆叠
- 顶部导航 sticky + 吸附效果

## 组件风格
- 卡片：半透明背景 + 背景模糊（玻璃拟态）
- 按钮：渐变边框、悬停亮度提升
- 标题：大字号、分层排版、留白空间
- 卡片阴影：柔和、边缘渐变

## 动效与交互
- 元素进入：淡入、上移、错落出现
- 悬停：轻微放大、阴影变化、背景高亮
- 滚动：视差背景或卡片错层
- 过渡：`ease-out`、`duration-300` / `duration-500`
- 弹性：小范围 `scale` / `translateY` 动画

## 内容交互
- 首页同时支持中英文切换按钮
- 简历页提供独立语言切换开关
- 页面内显示编辑按钮，进入“编辑模式”后可直接修改文本
- 账户或发布机制可后续补充，当前阶段先实现“本地编辑 + 保存草稿”
