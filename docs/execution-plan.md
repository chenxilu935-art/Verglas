# 开发执行步骤

## 1. 项目初始化
- 在 `个人网页` 文件夹内初始化 React + Vite 项目
- 安装 Tailwind CSS
- 建立基础目录：`src/pages/`、`src/components/`、`src/data/`
- 创建 `docs/` 和 `dev-logs/` 目录用于规范与记录

## 2. 搭建首页框架
- 完成首页静态结构：导航、Banner、精选卡片、语言切换
- 实现蓝紫色渐变背景与玻璃拟态卡片
- 先完成移动端布局，再补大屏错落网格
- 添加基础滚动吸附和微交互

## 3. 简历页与语言切换
- 构建简历页路由与布局
- 导入中英文简历内容（`Chenxi Lu`、`路晨曦3`）
- 增加按钮切换中英文显示
- 使用卡片/栅格展示教育、经验、技能等内容

## 4. 编辑模式与可编辑内容
- 在页面中添加“编辑”入口
- 实现可编辑文本区域或表单字段
- 支持本地图片占位上传，并展示头像预览
- 保持数据结构清晰，方便后续扩展保存机制

## 5. 项目页与联系页
- 构建项目页布局，支持后续项目内容展示
- 添加联系页与社交链接模块
- 继续优化视差、错落动画、响应式体验

## 6. 迭代与稳定
- 每次完成一项功能后测试响应式与兼容性
- 对动效与交互进行轻量优化
- 保持开发日志更新，定期总结完成项与待办项

## 7. 发布准备
- 做好项目目录清晰、文档齐全
- 保持版本控制提交说明
- 确保首页与简历页基础可用后再做扩展功能
