# 开发执行步骤

## 说明
每完成一个阶段的全部任务，再进入下一阶段。每个阶段结束后需通过 `vite build` 验证无报错，并在 `dev-logs/` 中记录完成情况。

---

## 阶段 1：编辑志排版基础升级 ✅
**目标**：不改变功能逻辑，只升级视觉排版。（已完成 2026-05-21）

- [x] 1.1 调整排版尺度：标题更大、留白更多、层级更分明
- [x] 1.2 优化色彩：从"蓝紫玻璃"调整为"深海暗底 + 暖纸色内容区"
- [x] 1.3 导航栏微调：更轻、更透明的编辑志风格 header
- [x] 1.4 响应式检查：移动端排版不崩

**验证**：`vite build` 通过；页面视觉明显更"编辑志"

---

## 阶段 2：明信片纸质化重设计 ✅
**目标**：明信片从"玻璃卡片"变为"真实纸质感漂流物"。（已完成 2026-05-21）

- [x] 2.1 添加 CSS 纸质纹理（噪点叠加 + 微颗粒 + 米白底色）
- [x] 2.2 优化漂浮动画：更自然的不规则漂动，模拟"从远方缓缓漂来"
- [x] 2.3 调整明信片分布：4 张分散排布，互不重叠，各有不同节奏
- [x] 2.4 增强水影效果：明信片下方光晕模拟水面反射
- [x] 2.5 每张明信片关联内容入口（介绍/故事/经历/项目）

**验证**：明信片有纸感，动画自然，不重叠 ✅

---

## 阶段 3：电影感滚动叙事 ✅
**目标**：滚动页面时内容有节奏地逐层揭示。（已完成 2026-05-21）

- [x] 3.1 建立统一的滚动揭示系统（staggered reveal + animation-delay）
- [x] 3.2 首页各内容区块配置独立的"登场"动画序列
- [x] 3.3 视差层次：海洋背景 / 明信片 / 内容卡片 以不同速度滚动
- [x] 3.4 滚动节奏微调：确保动画不卡顿、不过度

**验证**：滚动时内容逐层揭示，有电影感节奏 ✅

---

## 阶段 4：CSS 3D 景深效果 ✅
**目标**：用 CSS 3D 增强空间感和沉浸感。（已完成 2026-05-21）

- [x] 4.1 明信片 3D 翻转：保持现有 flip 效果，优化 3D 视角
- [x] 4.2 多层景深：不同元素赋予不同 z-index 和视差速度
- [x] 4.3 明信片 3D 倾斜：hover 时跟随鼠标微倾斜（tilt effect）
- [x] 4.4 性能检查：确保 CSS 3D 不过度消耗性能

**验证**：页面有明显的空间深度感 ✅

---

## 阶段 5：整体打磨与响应式 ✅
**目标**：统一风格、修复细节、确保多端可用。（已完成 2026-05-21）

- [x] 5.1 全页色彩和排版一致性检查
- [x] 5.2 移动端适配：明信片在小屏的尺寸和位置调整
- [x] 5.3 中英文切换流畅度检查
- [x] 5.4 编辑模式在新风格下的兼容调整
- [x] 5.5 性能测试：动画帧率、图片加载

**验证**：所有页面在手机/平板/桌面均可正常使用 ✅
