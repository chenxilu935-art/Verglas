# 项目需求文档

## 目标
- 搭建一个个人展示网站，支持 `React + Tailwind CSS`。
- 主页和其他页面并存；首页同时支持中英文切换。
- 简历页提供中英简历切换按钮。
- 页面内容可编辑，后续可扩展为上传照片、替换文本等功能。
- 风格为 `玻璃拟态 + 杂志编辑风`，以 `蓝色 + 紫色渐变` 为主色调。

## 核心页面
- 首页（Landing）
- 简历页（Resume）
- 项目页（Projects）
- 联系页（Contact）

## 视觉与交互
- 玻璃拟态卡片与模糊背景
- 流体布局与响应式栅格
- 滚动吸附导航
- 悬停反馈与微交互
- 滚动出现、错落动画、平滑过渡、视差滚动、弹性动画
- 轻颗粒背景与细腻质感

## 内容与语言
- 首页和简历页支持中英文切换
- 支持用户在页面内直接编辑文本与资料
- 后续可添加照片上传、项目介绍文本、社交链接等内容

## 质量目标
- 稳定、安全、有效推进开发
- 先建立基础页面，再逐步迭代交互和编辑能力
- 开发过程保持每日记录与待办清单
