# 项目需求文档

## 目标
- 搭建一个个人展示网站，支持 `React + Tailwind CSS`。
- 暗色编辑志风 + 海洋主题融合，打造沉浸式个人品牌体验。
- 首页和其他页面并存；首页同时支持中英文切换。
- 页面内容可编辑，后续可扩展为上传照片、替换文本等功能。

## 整体风格方向
- **主调**：暗色编辑志风（Dark Editorial）—— 克制用色、大留白、强排版节奏
- **主题**：海洋漂流 —— 明信片如漂流瓶般缓缓漂来，逐层展开个人介绍、故事和经历
- **质感**：真实纸纹理明信片 + CSS 3D 景深 + 电影感滚动叙事

## 核心页面
- 首页（Landing）— 海洋背景 + 漂流明信片 + 编辑志排版
- 简历页（Resume）— 卡片/栅格展示教育、经验、技能
- 项目页（Projects）
- 联系页（Contact）

## 视觉与交互
- 真实海洋照片背景 + 暗角叠加
- 明信片：纸质纹理质感，缓缓漂流动效（如漂流瓶）
- 电影感滚动叙事：staggered reveal、scroll-triggered animation
- CSS 3D 效果：卡片翻转、景深感、视差层次
- 编辑志排版：大字号标题、不对称布局、节奏感留白
- 轻颗粒背景与细腻质感

## 内容与语言
- 首页和简历页支持中英文切换
- 支持用户在页面内直接编辑文本与资料
- 后续可添加照片上传、项目介绍文本、社交链接等内容

## 字体
- 标题：楷体（优先使用系统楷体 KaiTi）
- 正文：Arial / sans-serif

## 质量目标
- 稳定、安全、有效推进开发
- 按阶段推进，每个阶段完成一个清晰可测试的交付物
- 先建立基础页面，再逐步迭代交互和编辑能力
- 开发过程保持每日记录与待办清单
